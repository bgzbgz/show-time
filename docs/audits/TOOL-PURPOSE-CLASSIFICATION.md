# Tool Purpose Classification Report

> Generated from one-by-one logic audit comparing each tool's HTML implementation against its sprint content.md

## Classification Types

| Type | Definition | Example |
|------|-----------|---------|
| **Reflection** | Guided self-assessment with qualitative text inputs; introspective | Know Thyself |
| **Brainstorm** | Idea generation with listing, ranking, or clustering | Goals (Impact/Easy) |
| **Calculator** | Numeric inputs producing computed outputs (formulas, scores) | Cash Flow, Market Size |
| **Assessment** | Structured evaluation with scoring/rating scales | Team (Lencioni), FIT ABC |
| **Canvas** | Multi-section strategic framework filled in step-by-step | Dream, Value Proposition |
| **Planning** | Action-oriented with owners, timelines, milestones | Meeting Rhythm, Implementation Plans |
| **Validation** | Testing hypotheses against criteria/checklists | VP Testing, VP Stress Test |
| **Builder** | Creates a concrete deliverable (statement, charter, playbook) | Brand Marketing, Employer Branding |
