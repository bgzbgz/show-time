# CLIENT PROFILE SUMMARY
## Quick Reference for Tool Psychology Audits

---

## CORE IDENTITY: THE IMPRISONED ATHLETE

| Trait | Description |
|-------|-------------|
| **4:47 AM CEO** | Wakes 13 minutes before "everyone else" — needs to be measurably superior |
| **System Architect** | Builds systems to prevent problems, not solve them. Reactive mode = personal failure |
| **Information Paradox** | 47 browser tabs open. Over-researched, under-decisive. Craves curation |
| **Metric Machine** | Quantifies everything. If it can't be measured, it doesn't exist |
| **Imprisoned Champion** | Former star performer trapped by complexity they built |

**Core Wound:** Self-worth = today's metrics / yesterday's metrics. Using achievement to fill existential void.

**Demographics:** 38-52 years old, $10M-$250M revenue, 50-500 people, ex-top performer or founder.

## LANGUAGE REQUIREMENTS

**POWER WORDS (Use Liberally):**
Surgical, Systematic, Relentless, Execute, Dominate, Brutal, Raw, Unfiltered, Sharp, Precise, Elite, Peak, Maximum, Exponential, Transform, Architect, Engineer, Build, Scale, Optimize

**FORBIDDEN WORDS (Never Use):**
Journey, Process, Holistic, Comprehensive, Leverage, Solutions, Methodologies, Frameworks, Best Practices, Collaboration, Partnership, Transformation (without metrics), Work-life balance, Wellness, Mindfulness

**Sentence Architecture:** [SHOCKING TRUTH] + [SPECIFIC DATA] + [IMMEDIATE IMPLICATION]
- Max 8-12 words per sentence
- Staccato delivery
- Period dominance (certainty, not complexity)
- Active voice only

**Paragraph Structure:**
1. Brutal truth (hook)
2. Specific evidence (proof)
3. Direct consequence (urgency)
4. Clear next action (direction)

## CONTENT RULES

1. **30-Second Rule:** Deliver value in 30 seconds or lose them forever
2. **No filler, no fluff, no "welcome" videos**
3. **Start with the most valuable insight**
4. **80% implementation, 20% theory**
5. **Bullet points over paragraphs**
6. **Bold key insights for scanning**
7. **90-minute maximum for any engagement**

**CTA Language:**
- Never "Learn more" -> Always "Start now"
- Never "Consider joining" -> Always "Claim your spot"
- Never "Free consultation" -> Always "45-minute implementation session"

---

## THE 4-QUESTION TEST (Every Piece of Content)

1. Would this make a 4:47 AM CEO stop scrolling?
2. Does it respect their intelligence while offering genuine insight?
3. Can they implement this immediately for competitive advantage?
4. Does it make them feel like the elite performer they once were?

**Final Filter:** "Would this make them think: 'Finally, someone who gets it'?"

---

## REQUIRED TOOL ELEMENTS

1. **Fast Track Insight Boxes (Yellow)** — 1 per step, "80/20 hack" or "brutal truth"
2. **Brutal Honesty Prompts** — "CEOs overestimate X by 2x. Be brutally honest."
3. **Competitive Framing** — "While competitors do X, you'll do Y"
4. **Time Compression Language** — "72 hours", "90 days not 2 years"
5. **Peer Proof/Social Validation** — CEO names, specific companies, results
6. **Metric/ROI Framing** — Specific numbers, percentages, multipliers
7. **Athletic/Military Metaphors** — "elite athlete", "architect", "engineer", "dominate"
8. **Anti-Generic Positioning** — "For CEOs who refuse mediocrity"
