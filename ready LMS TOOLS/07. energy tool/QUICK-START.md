- Check that `fonts/` folder is in the same directory as HTML
- File names are case-sensitive on Mac/Linux
- Try refreshing browser (Cmd/Ctrl + Shift + R)

- Don't use incognito/private mode
- Check browser console (F12) for errors
- "Last saved" timestamp should update every 2 seconds

### Submission Failed
- Check internet connection
- Verify webhook URL in HTML file (line 256)
