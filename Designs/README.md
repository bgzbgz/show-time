# Designs/

Brand assets for the Fast Track platform. Source of truth for all visual identity.

## Structure

| Folder | Contents |
|---|---|
| `00. brand guidelines/` | Fast Track Brand Guidelines master doc |
| `01. favicon/` | Favicon PNG (16x16) |
| `02. logos/` | FastTrack F logo (white PNG) |
| `03. Fonts/` | Font files in two formats: `opentype/` (.otf) and `woff2/` (.woff2) |
| `04. TOV/` | Tone of Voice master guide |
| `05. images for tools/` | Hero/cover images used in tool cover screens |

## Font files

Four fonts used across the platform:

| Font | Use |
|---|---|
| `Plaak3Trial-43-Bold` | Headers, titles, large numbers |
| `RiformaLL-Regular` | Body text, paragraphs |
| `RangleRiformaWeb-Medium` | Medium weight body text |
| `MonumentGrotesk-Mono` | Labels, badges, monospace elements |

The `.woff2` files here are the master copies. They are also duplicated into `frontend/shared/fonts/` and into each tool module folder for production serving.

## Brand colors

- Black `#000000` — primary background
- White `#FFFFFF` — text and backgrounds
- Yellow `#FFF469` — accent, CTAs, highlights
