# Sprint 00: WOOP - Quick Reference

## Purpose
Prepare participants for the Fast Track journey by introducing key preparation tools and learning methodologies for the entire program.

## Decision to Make
How will you approach this transformational program to maximize learning and impact?

## Key Concepts
- **WOOP Technique**: Wish, Outcome, Obstacle, Plan framework for goal achievement
- **Pre-Mortem Analysis**: Anticipating and preventing potential failures
- **Fast Track Learning**: Active recall and spaced repetition methodologies

## Tool Sections
1. Tune-In Questions - Set expectations and mindset
2. Brain Juice - Familiarize with program structure
3. Think and Do - Practice WOOP and Pre-Mortem tools
4. Team Session - Align on program approach

## Dependencies
- Requires: None (Introduction sprint)
- Feeds into: All subsequent sprints (foundational preparation)
