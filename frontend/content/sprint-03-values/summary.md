# Sprint 03: Values - Quick Reference

## Purpose
Define and embed core company values that shape culture, guide decisions, and drive organizational behavior.

## Decision to Make
What are the non-negotiable values that will serve as our company's compass for all decisions and actions?

## Key Concepts
- **Core Values Definition**: Identifying the 3-5 values that matter most
- **Values Integration**: Embedding values in hiring, performance, and decisions
- **Cultural Foundation**: Using values to build strong, cohesive culture

## Tool Sections
1. Review of Last Sprint - Building on Dream
2. Reading Materials - Values frameworks and examples
3. Think and Do - Define actionable company values
4. Team Session - Align on values and implementation

## Dependencies
- Requires: Sprint 02 (Dream)
- Feeds into: Sprint 04 (Team), Sprint 25 (Org Design), Sprint 26 (Employer Branding)
